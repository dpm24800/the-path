- What is Data Structure?
- [Data Structure](https://www.geeksforgeeks.org/data-structure/)
- Data Structure is a way of organizing data in a computer so that it can be used efficiently. It is a collection of data values, relationship(s) between data items, and functions that can be applied to the data. It is a way of storing data in a computer so that it can be used efficiently. 

- Way to store and organize the data so that it can be accessed effectively and also it can be used and manipulated in the future easily.
- Data Structure is basic concept of any programming language.
